# Stotra (Devotional Hymns)

Guide to Sanskrit devotional literature: hymns, prayers, and sacred poetry.

## Types of Stotras

| Type | Meaning | Structure |
|------|---------|-----------|
| **स्तोत्रम्** | Hymn of praise | Variable verses |
| **अष्टकम्** | Eight verses | 8 verses + phala shruti |
| **शतकम्** | Hundred verses | 100+ verses |
| **सहस्रनाम** | Thousand names | 1000 divine names |
| **कवचम्** | Armor | Protection prayer |
| **स्तुति** | Eulogy | Praise verses |
| **सुप्रभातम्** | Morning prayer | Wake-up hymn |
| **पञ्चरत्नम्** | Five jewels | 5 verses |
| **मङ्गलम्** | Auspicious | Blessing verses |
| **चालीसा** | Forty verses | 40 verses (Hindi influence) |

## Stotra Structure

### Typical Pattern

1. **आवाहन / ध्यान** — Invocation/meditation verse
2. **मूल स्तोत्र** — Main hymn body
3. **फलश्रुति** — Benefits of recitation
4. **समर्पण** — Offering/dedication

### Phala Shruti (Benefits)

Most stotras end with verses describing:
- Spiritual benefits (मोक्ष, भक्ति)
- Material benefits (धन, सन्तान, आरोग्य)
- Protection (रक्षा, शत्रुनाश)
- Conditions (how/when to recite)

## Recitation Guidelines

### Sankalpa (Resolution)

Before reciting, state:
- Date (तिथि, वार, नक्षत्र)
- Location (देश, स्थान)
- Purpose (कामना)

### Common Practices

| Practice | Description |
|----------|-------------|
| जप | Repetition with counting |
| पारायण | Complete recitation |
| अनुष्ठान | Prescribed number over days |
| नित्यपाठ | Daily recitation |

### Auspicious Times

| Time | Suitable For |
|------|--------------|
| ब्राह्ममुहूर्त | All stotras |
| प्रातः | सुप्रभातम्, आदित्यहृदयम् |
| सन्ध्या | गायत्री, सन्ध्यावन्दन |
| प्रदोष | शिव stotras |
| अर्धरात्रि | Devi stotras (special occasions) |
