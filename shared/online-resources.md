# Sanskrit Online Resources

## Recommended Platforms

### Ambuda (ambuda.org)

**Volunteer-run digital Sanskrit library**

| Feature | Description |
|---------|-------------|
| **Texts** | Vedas, Epics (Mahabharata, Ramayana), Kavyas, Stotras |
| **Authors** | Kalidasa, Bhavabhuti, and classical poets |
| **Dictionaries** | Integrated lookup tools |
| **Community** | Volunteer proofing, GitHub access, Discord |
| **Languages** | Interface in English, Sanskrit, Hindi, Marathi, Telugu |
| **Special** | "Sponsor a book" program, open-source |

### Sanskrit Sahitya (sanskritsahitya.org)

**AI-powered Sanskrit literature platform**

| Feature | Description |
|---------|-------------|
| **Texts** | Mahakavyas, Khandakavyas, Natakas, Puranas, Kathas |
| **Commentaries** | Mallinatha and other traditional tikas |
| **AI Search** | Semantic search — find shlokas by meaning |
| **Word Analysis** | Padachheda (word splitting) + dictionary lookup |
| **Chandas** | Automatic meter identification |
| **Scripts** | Devanagari, IAST, multiple display options |
| **Offline** | Download texts for offline use |
| **Open Data** | All data free on GitHub |

### Ashtadhyayi.com (ashtadhyayi.com)

**Comprehensive Paninian grammar portal**

| Feature | Description |
|---------|-------------|
| **Sutra Patha** | All 4000 sutras with navigation |
| **Dhatu Patha** | Verbal roots with derivations |
| **Prakriya Generator** | Step-by-step word derivation |
| **Shabda Patha** | Noun/nominal paradigms |
| **Kosha** | Dictionary search |
| **Audio** | Chanting with adjustable speed |
| **Courses** | Structured online learning |
| **Transliteration** | IAST, ITRANS, SLP1, Harvard-Kyoto |
| **Offline** | Download for offline study |

### Dharmamitra (dharmamitra.org)

**AI-powered Dharma text toolkit**

| Feature | Description |
|---------|-------------|
| **Translation** | Machine translation: Sanskrit, Tibetan, Buddhist Chinese, English |
| **MITRA Search** | 2,500 years of Buddhist texts — keyword + semantic search |
| **Grammar Analysis** | Sanskrit parsing with lemmatization, word-level analysis |
| **OCR** | Extract text from scans (Devanagari, Tibetan, Chinese) |
| **Scripts** | Supports IAST, Harvard-Kyoto, Wylie, Devanagari |
| **Browser Extension** | Chrome/Firefox for in-context translation |
| **Special** | Research and "explained" translation modes |
