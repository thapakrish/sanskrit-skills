# Stotra Composers

## Major Composers

### शङ्कराचार्य (Shankaracharya) — 788-820 CE

**Title**: जगद्गुरु (World Teacher)
**Philosophy**: Advaita Vedanta
**Significance**: Most prolific stotra composer; established four mathas

#### Stotras by Deity

| Deity | Stotras |
|-------|---------|
| **विष्णु** | गोविन्दाष्टकम्, अच्युताष्टकम्, भजगोविन्दम्, जगन्नाथाष्टकम्, पाण्डुरङ्गाष्टकम् |
| **शिव** | शिवपञ्चाक्षरस्तोत्रम्, लिङ्गाष्टकम्, बिल्वाष्टकम्, द्वादशज्योतिर्लिङ्गम्, दक्षिणामूर्तिस्तोत्रम्, शिवानन्दलहरी |
| **देवी** | सौन्दर्यलहरी, भवान्यष्टकम्, अन्नपूर्णास्तोत्रम्, कनकधारास्तोत्रम्, महिषासुरमर्दिनी, मीनाक्षीपञ्चरत्नम्, त्रिपुरसुन्दर्यष्टकम् |
| **गणेश** | गणेशपञ्चरत्नम् |
| **हनुमान्** | हनुमत्पञ्चरत्नम् |
| **आत्मज्ञान** | निर्वाणषट्कम्, आत्मषट्कम्, मनीषापञ्चकम् |
| **सन्न्यास** | काशीपञ्चकम्, मायापञ्चकम् |

#### Characteristics
- Philosophical depth
- Compact, powerful verses
- Often uses "अष्टक" (8-verse) or "पञ्चरत्न" (5-verse) format
- Balances भक्ति and ज्ञान

### वेदान्तदेशिक (Vedanta Desika) — 1268-1369 CE

**Philosophy**: Vishishtadvaita
**Location**: Tamil Nadu (Kanchipuram, Srirangam)

| Stotra | Description |
|--------|-------------|
| **श्रीस्तुति** | Praise of Lakshmi |
| **भगवद्ध्यानसोपानम्** | Meditation ladder |
| **गोपालविंशति** | 20 verses on Krishna |
| **देवनायकपञ्चाशत्** | 50 verses on Vishnu |
| **अभीतिस्तव** | Freedom from fear |
| **दयाशतकम्** | 100 verses on mercy |
| **पादुकासहस्रम्** | 1000 verses on Lord's sandals |

### आळ्वार्गण (Alvars) — 6th-9th century CE

**Tradition**: Tamil Vaishnavism
**Works**: नालायिरदिव्यप्रबन्धम् (4000 verses in Tamil)

| Alvar | Famous Work |
|-------|-------------|

| कुलशेखर | मुकुन्दमाला (Sanskrit), पेरुमाळ्तिरुमोऴि |

### पुष्पदन्त (Pushpadanta) — ~10th century CE

**Identity**: Gandharva (celestial musician)
**Famous work**: शिवमहिम्नःस्तोत्रम्

**Legend**: Lost his powers after stepping on bilva leaves; composed the stotra to regain Shiva's grace.

---

### मूककवि (Muka Kavi) — ~15th century CE

**Location**: Kanchipuram
**Legend**: Born mute; gained speech through Kamakshi's grace

| मन्दस्मितशतकम् | 100 | Gentle smile |
| कटाक्षशतकम् | 100 | Sidelong glance |

## Stotra Characteristics by Composer

| Composer | Style |
|----------|-------|
| Shankaracharya | Philosophical, compact, powerful |
| Pushpadanta | Poetic, ornate, devotional |
| Vedanta Desika | Elaborate, scholarly, devotional |
| Muka Kavi | Sensuous descriptions, tantric |
| Tulsidas | Simple, devotional, accessible |
| Vallabhacharya | Sweet, loving, intimate |
