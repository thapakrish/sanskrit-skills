# Devi Stotras

## सौन्दर्यलहरी (Soundarya Lahari)

**Author**: शङ्कराचार्य
**Verses**: 103 (41 आनन्दलहरी + 62 सौन्दर्यलहरी)
**Theme**: Divine beauty of the Goddess; Shri Vidya tantra

### Structure
- Verses 1-41: आनन्दलहरी (Wave of Bliss) — philosophical, tantric
- Verses 42-103: सौन्दर्यलहरी (Wave of Beauty) — descriptive, devotional

### Opening Verse

```
शिवः शक्त्या युक्तो यदि भवति शक्तः प्रभवितुम्।
न चेदेवं देवो न खलु कुशलः स्पन्दितुमपि।
अतस्त्वामाराध्यां हरिहरविरिञ्चादिभिरपि
प्रणन्तुं स्तोतुं वा कथमकृतपुण्यः प्रभवति॥ १॥
```

*"Shiva becomes capable of creating only when united with Shakti; otherwise, He cannot even stir. Therefore, how can one without merit dare to bow to or praise You, who are worshipped even by Hari, Hara, and Brahma?"*

### Famous Verses

**Verse 44** (Beauty description):
```
तनीयांसं पांसुं तव चरणपङ्केरुहभवं
विरिञ्चिस्सञ्चिन्वन् विरचयति लोकानविकलम्।
```

*"Brahma creates the worlds using the pollen-dust from Your lotus feet."*

## ललितासहस्रनाम (Lalita Sahasranama)

**Source**: ब्रह्माण्डपुराण
**Names**: 1000 names of Lalita Tripurasundari

### ध्यान श्लोक

```
सिन्दूरारुणविग्रहां त्रिनयनां माणिक्यमौलिस्फुरत्
तारानायकशेखरां स्मितमुखीमापीनवक्षोरुहाम्।
पाणिभ्यामलिपूर्णरत्नचषकं रक्तोत्पलं बिभ्रतीं
सौम्यां रत्नघटस्थरक्तचरणां ध्यायेत्परामम्बिकाम्॥
```

### Opening Names

```
श्रीमाता श्रीमहाराज्ञी श्रीमत्सिंहासनेश्वरी।
चिदग्निकुण्डसम्भूता देवकार्यसमुद्यता॥
```

## महिषासुरमर्दिनीस्तोत्रम् (Mahishasura Mardini Stotram)

**Author**: शङ्कराचार्य (attributed)
**Theme**: Devi slaying the buffalo demon

### Selected Verses

```
अयि गिरिनन्दिनि नन्दितमेदिनि विश्वविनोदिनि नन्दिनुते
गिरिवरविन्ध्यशिरोऽधिनिवासिनि विष्णुविलासिनि जिष्णुनुते।
भगवति हे शितिकण्ठकुटुम्बिनि भूरिकुटुम्बिनि भूरिकृते
जय जय हे महिषासुरमर्दिनि रम्यकपर्दिनि शैलसुते॥ १॥

सुरवरवर्षिणि दुर्धरधर्षिणि दुर्मुखमर्षिणि हर्षरते
त्रिभुवनपोषिणि शङ्करतोषिणि किल्बिषमोषिणि घोषरते।
दनुजनिरोषिणि दितिसुतरोषिणि दुर्मदशोषिणि सिन्धुसुते
जय जय हे महिषासुरमर्दिनि रम्यकपर्दिनि शैलसुते॥ २॥
```

### Characteristics
- Refrain: जय जय हे महिषासुरमर्दिनि रम्यकपर्दिनि शैलसुते
- Complex rhyming patterns
- Vigorous rhythm (suitable for recitation during Navaratri)

## कनकधारास्तोत्रम् (Kanakadhara Stotram)

**Author**: शङ्कराचार्य
**Legend**: Composed when a poor woman could only offer Shankara a single amla fruit; Lakshmi showered gold (कनकधारा)

### Selected Verses

```
अङ्गं हरेः पुलकभूषणमाश्रयन्ती
भृङ्गाङ्गनेव मुकुलाभरणं तमालम्।
अङ्गीकृताखिलविभूतिरपाङ्गलीला
माङ्गल्यदाऽस्तु मम मङ्गलदेवतायाः॥ १॥
```

## देव्यपराधक्षमापनस्तोत्रम् (Forgiveness Stotra)

**Author**: शङ्कराचार्य

```
न मन्त्रं नो यन्त्रं तदपि च न जाने स्तुतिमहो
न चाह्वानं ध्यानं तदपि च न जाने स्तुतिकथाः।
न जाने मुद्रास्ते तदपि च न जाने विलपनं
परं जाने मातस्त्वदनुसरणं क्लेशहरणम्॥
```

*"I know no mantra, no yantra, no praise, no invocation, no meditation, no mudras, no lament—but I know, O Mother, that following You removes all sorrow."*

## भवान्यष्टकम् (Bhavanyashtakam)

**Author**: शङ्कराचार्य

```
न तातो न माता न बन्धुर्न दाता न पुत्रो न पुत्री न भृत्यो न भर्ता।
न जाया न विद्या न वृत्तिर्ममैव गतिस्त्वं गतिस्त्वं त्वमेका भवानि॥ १॥
```

*"No father, no mother, no relative, no giver, no son, no daughter, no servant, no master, no wife, no learning, no livelihood—You alone are my refuge, O Bhavani."*

## अन्नपूर्णास्तोत्रम् (Annapurna Stotram)

**Author**: शङ्कराचार्य
**Theme**: Goddess as provider of food (अन्न)

```
नित्यानन्दकरी वराभयकरी सौन्दर्यरत्नाकरी
निर्धूताखिलघोरपावनकरी प्रत्यक्षमाहेश्वरी।
प्रालेयाचलवंशपावनकरी काशीपुराधीश्वरी
भिक्षां देहि कृपावलम्बनकरी मातान्नपूर्णेश्वरी॥ १॥
```

## मीनाक्षीपञ्चरत्नम् (Meenakshi Pancharatnam)

**Author**: शङ्कराचार्य
**Theme**: Meenakshi of Madurai

```
उद्यद्भानुसहस्रकोटिसदृशां केयूरहारोज्ज्वलां
बिम्बोष्ठीं स्मितदन्तपङ्क्तिरुचिरां पीताम्बरालङ्कृताम्।
विष्णुब्रह्मसुरेन्द्रसेवितपदां तत्त्वस्वरूपां शिवां
मीनाक्षीं प्रणतोऽस्मि सन्ततमहं कारुण्यवारांनिधिम्॥
```

## दुर्गासूक्तम् (Durga Suktam)

**Source**: Vedic (Taittiriya Aranyaka)

```
जातवेदसे सुनवाम सोममरातीयतो निदहाति वेदः।
स नः पर्षदति दुर्गाणि विश्वा नावेव सिन्धुं दुरितात्यग्निः॥
```
