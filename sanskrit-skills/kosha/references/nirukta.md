# Nirukta (Etymology)

Word derivations following traditional Sanskrit etymological analysis.

---

## Principles of Nirukta

The science of etymology (निरुक्त) explains word meanings through:
1. **Dhatu** (root) identification
2. **Pratyaya** (suffix) analysis
3. **Upasarga** (prefix) effects
4. **Yoga** (derivational meaning) vs **Rudhi** (conventional meaning)

## Divine Names

### ब्रह्मा
| Name | Derivation |
|------|------------|
| ब्रह्मा | √बृह् (to grow) + मनिन् → "the one who makes grow" |
| विधाता | वि + √धा (to place) + तृच् → "the arranger/ordainer" |
| विधिः | वि + √धा + कि → "rule, fate, creator" |
| स्वयम्भूः | स्वयम् + √भू → "self-existent" |
| पितामहः | पितृ + महत् → "great father, grandfather" |
| चतुराननः | चतुर् + आनन → "four-faced" |
| कमलासनः | कमल + आसन → "lotus-seated" |
| हिरण्यगर्भः | हिरण्य + गर्भ → "golden-wombed" |

### विष्णुः
| Name | Derivation |
|------|------------|
| विष्णुः | √विष् (to pervade) + नु → "the pervader" |
| नारायणः | नार (water) + अयन → "abode of waters" OR नर + अयन → "goal of humans" |
| हरिः | √हृ (to take away) + इ → "remover (of sins)" |
| गोविन्दः | गो + √विद् → "cow-finder" OR "sense-knower" |
| माधवः | मधु + अण् → "descendant of Madhu" OR "spring-related" |
| केशवः | क (Brahma) + ईश → "lord of Brahma" OR केश + व → "having beautiful hair" |
| दामोदरः | दाम (rope) + उदर → "rope-bellied" (tied by Yashoda) |
| पुण्डरीकाक्षः | पुण्डरीक + अक्षि → "lotus-eyed" |
| चक्रपाणिः | चक्र + पाणि → "discus-handed" |
| पद्मनाभः | पद्म + नाभि → "lotus-naveled" |
| जनार्दनः | जन + √अर्द् → "troubler of people" (the wicked) |
| वासुदेवः | वसुदेव + अण् → "son of Vasudeva" |
| अच्युतः | न + च्युत → "unfallen, imperishable" |

## Nature Terms

### Elements

| Word | Derivation |
|------|------------|
| जलम् | √जल् (to be sharp/cool) → "the cooling one" |
| वारि | √वृ (to cover) + इन् → "the covering one" |
| उदकम् | उद् + √अक् (to go) → "that which goes up" (evaporates) |
| अम्बु | √अम्ब् (to go) → "the moving one" |
| सलिलम् | √सृ (to flow) + इलच् → "the flowing one" |
| अग्निः | √अञ्ज् (to anoint) OR अग्र + √नी → "leader" |
| पावकः | √पू (to purify) + ण्वुल् → "the purifier" |
| अनलः | अन् (to breathe) + √अल् → "not enough" (for breath/life) |
| वायुः | √वा (to blow) + उ → "the blower" |
| पवनः | √पू (to purify) + युच् → "the purifier" |

### Celestial

| Word | Derivation |
|------|------------|
| आकाशः | आ + √काश् (to shine) → "that which shines forth" |
| गगनम् | √गम् (to go) + युच् → "place of going" (for birds, gods) |
| नभः | √नभ् (to burst) → "the expansive" |
| खम् | √खन् (to dig) → "the excavated, empty space" |
| द्यौः | √द्युत् (to shine) → "the shining one" |

### Earth & Geography

| Word | Derivation |
|------|------------|
| पृथ्वी | पृथु (broad) + ङीप् → "the broad one" |
| भूमिः | √भू (to be) + मि → "place of being" |
| धरा | √धृ (to hold) + टाप् → "the holder" |
| वसुधा | वसु (wealth) + √धा → "wealth-holder" |
| मेदिनी | मेदस् (fat) + इनि → "the fat/fertile one" |
| पर्वतः | पर्वन् (joint/knot) + अतच् → "having joints/sections" |
| गिरिः | √गॄ (to swallow) + इ → "the swallower" (of water) |
| समुद्रः | सम् + उद् + √द्रु → "where waters flow together" |
| सागरः | √सृ (to flow) + घञ् → "containing flowing waters" |
| नदी | √नद् (to sound) + ई → "the sounding one" |

## Human Terms

### Body Parts

| Word | Derivation |
|------|------------|
| शिरः | √शॄ (to break) → "that which can be broken" (vulnerable) |
| मुखम् | √मुच् (to release) → "that which releases" (speech, breath) |
| नेत्रम् | √नी (to lead) + ष्ट्रन् → "that which leads" (guides sight) |
| चक्षुः | √चक्ष् (to see) + उस् → "the seer" |
| कर्णः | √कृ (to do) OR √कॄ (to pour) → "that which does/pours in" (sound) |
| हस्तः | √हस् (to seize) → "the seizer" |
| पादः | √पद् (to go) + घञ् → "the goer" |
| हृदयम् | √हृ (to take) + दय → "that which takes" (blood in/out) |
| जिह्वा | √जिह्व (to lick) → "the licker" |

### Relationships

| Word | Derivation |
|------|------------|
| पिता | √पा (to protect) + तृच् → "the protector" |
| माता | √मा (to measure) + तृच् → "the measurer" (of food, care) |
| पुत्रः | पुन् (hell) + √त्रै → "savior from hell (Put)" |
| भ्राता | √भृ (to bear) + तृच् → "the supporter" |
| पत्नी | पति + ङीप् → "wife of lord" |
| दुहिता | √दुह् (to milk) + तृच् → "the milker" (of father's wealth) |

### States & Qualities

| Word | Derivation |
|------|------------|
| सुखम् | सु + खम् → "good space" (for breath, movement) |
| दुःखम् | दुस् + खम् → "bad space" (constriction) |
| आनन्दः | आ + √नन्द् (to rejoice) → "complete rejoicing" |
| शोकः | √शुच् (to grieve) + घञ् → "grief" |
| क्रोधः | √क्रुध् (to be angry) + घञ् → "anger" |
| मोहः | √मुह् (to be confused) + घञ् → "delusion" |
| ज्ञानम् | √ज्ञा (to know) + ल्युट् → "knowledge" |
| विद्या | √विद् (to know) + यत् + टाप् → "knowledge" |

## Philosophical Terms

| Term | Derivation | Meaning |
|------|------------|---------|
| आत्मा | √अत् (to move) / √अप् (to pervade) / √आ (to breathe) | the self, that which pervades |
| ब्रह्म | √बृह् (to grow) | the great, the absolute |
| मायया | √मा (to measure) + या | that which measures out (illusion) |
| कर्म | √कृ (to do) + मन् | action |
| धर्मः | √धृ (to hold) + मन् | that which upholds |
| मोक्षः | √मुच् (to release) + घञ् | liberation |
| संसारः | सम् + √सृ (to flow) | flowing together, cycle |
| निर्वाणम् | निस् + √वा (to blow) | blown out, extinguished |
