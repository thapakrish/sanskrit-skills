# Linganushasana (Gender Rules)

Rules for determining grammatical gender (लिङ्ग) of Sanskrit nouns.

---

## The Three Genders

| Gender | Sanskrit | Abbreviation |
|--------|----------|--------------|
| Masculine | पुंल्लिङ्ग | पुं. |
| Feminine | स्त्रीलिङ्ग | स्त्री. |
| Neuter | नपुंसकलिङ्ग | न. / नपुं. |

---

## Masculine (पुंल्लिङ्ग)

### By Natural Gender
- Male beings: नरः, पुरुषः, राजा, देवः

### By Category

| Category | Examples | Notes |
|----------|----------|-------|
| **Mountains** | हिमालयः, विन्ध्यः, मेरुः | All mountain names |
| **Oceans** | समुद्रः, सागरः, अर्णवः | All ocean names |
| **Days of week** | रविवारः, सोमवारः | All weekdays |
| **Months** | चैत्रः, वैशाखः, आषाढः | All month names |
| **Trees** (individual) | वटः, अश्वत्थः, चन्दनः | When named as individuals |
| **Planets** | सूर्यः, चन्द्रः, मङ्गलः | Celestial bodies |
| **Winds** | वायुः, पवनः, मारुतः | |
| **Seasons** | वसन्तः, ग्रीष्मः, शिशिरः | Most season names |
| **Directions** | पूर्वः, दक्षिणः (as gods) | When personified |

### By Ending

| Ending | Examples | Exceptions |
|--------|----------|------------|
| **-अ** (majority) | रामः, देवः, वृक्षः | Many neuter -अ words |
| **-इ** (many) | कविः, मुनिः, अग्निः | Some feminine -इ |
| **-उ** (many) | गुरुः, रिपुः, भानुः | Some feminine -उ |
| **-तृ** (agent) | कर्ता, दाता, हर्ता | Feminine -त्री forms exist |

---

## Feminine (स्त्रीलिङ्ग)

### By Natural Gender
- Female beings: नारी, स्त्री, माता, कन्या

### By Category

| Category | Examples | Notes |
|----------|----------|-------|
| **Rivers** | गङ्गा, यमुना, सरस्वती | All river names |
| **Earth** | पृथ्वी, भूमिः, धरा | All earth synonyms |
| **Night** | रात्रिः, निशा, यामिनी | |
| **Directions** | पूर्वा, दक्षिणा | As abstract directions |
| **Creepers** | लता, वल्ली | Climbing plants |
| **Languages** | संस्कृता, प्राकृता | Language names |
| **Cities** (many) | अयोध्या, मथुरा | City names often feminine |
| **Scriptures** | श्रुतिः, स्मृतिः, गीता | Sacred texts |

### By Ending

| Ending | Examples | Notes |
|--------|----------|-------|
| **-आ** | रमा, सीता, गङ्गा | Almost always feminine |
| **-ई** | नदी, देवी, नारी | Almost always feminine |
| **-ऊ** | वधूः, चमूः | Mostly feminine |
| **-ति** (abstract) | गतिः, मतिः, शक्तिः | Abstract nouns |
| **-ता** (abstract) | सुन्दरता, मधुरता | Quality abstracts |
| **-त्री** (agent) | कर्त्री, दात्री | Female agents |
| **-आनी** | इन्द्राणी, रुद्राणी | Wives of gods |

---

## Neuter (नपुंसकलिङ्ग)

### By Category

| Category | Examples | Notes |
|----------|----------|-------|
| **Fruits** | फलम्, आम्रम्, कदलीफलम् | Almost all fruits |
| **Flowers** (many) | पुष्पम्, कुसुमम्, पद्मम् | Many flower names |
| **Metals** | सुवर्णम्, रजतम्, लोहम् | All metals |
| **Body parts** (many) | मुखम्, नेत्रम्, हृदयम् | Many (not all) |
| **Leaves/Grass** | पत्रम्, पर्णम्, तृणम् | |
| **Water** (forms) | जलम्, वारि, उदकम्, तोयम् | Most water words |
| **Abstract concepts** | सुखम्, दुःखम्, ज्ञानम् | Many abstracts |
| **Grains** | धान्यम्, व्रीहि | Many grains |
| **Countries** (some) | भारतम्, मगधम् | Some region names |

### By Ending

| Ending | Examples | Notes |
|--------|----------|-------|
| **-अम्** (many) | फलम्, वनम्, जलम् | Very common neuter ending |
| **-उ** (some) | मधु, वस्तु, अस्थि | Selected -उ words |
| **-इ** (some) | वारि, अक्षि | Selected -इ words |
| **-अस्** | मनस्, तपस्, तेजस् | -अस् stems |
| **-त्व** (abstract) | पुरुषत्वम्, सुन्दरत्वम् | -त्व abstracts |
| **-त्र** (instrument) | पात्रम्, छत्रम्, यन्त्रम् | Instruments |

---

## Suffix-Based Gender

### Always Feminine

| Suffix | Type | Example |
|--------|------|---------|
| -ता | तद्धित (abstract) | सुन्दरता (beauty) |
| -ति | कृत् (action) | गतिः (motion) |
| -त्री | कृत् (female agent) | कर्त्री (doer) |
| -आनी | तद्धित (wife of) | इन्द्राणी |

### Always Neuter

| Suffix | Type | Example |
|--------|------|---------|
| -त्व | तद्धित (abstract) | मनुष्यत्वम् (humanity) |
| -ल्युट् (-अन) | कृत् (action noun) | गमनम् (going) |
| -त्र | कृत् (instrument) | पात्रम् (vessel) |

### Generally Masculine

| Suffix | Type | Example |
|--------|------|---------|
| -तृ | कृत् (agent) | कर्ता (doer) |
| -अक | कृत् (agent) | नायकः (leader) |
| -इन् | तद्धित (possessor) | धनिन् (wealthy) |

## Dual-Gender Words

Some words can be multiple genders with different meanings:

| Word | Masculine | Feminine | Neuter |
|------|-----------|----------|--------|
| पद | — | — | पदम् (word, step) |
| लोक | लोकः (world) | — | — |
| आत्मन् | आत्मा (soul) | — | — |
| काल | कालः (time) | — | — |

---

## Special Cases

### Words with Optional Gender

| Word | Options | Meaning |
|------|---------|---------|
| मार्ग | पुं./न. | path |
| आसन | पुं./न. | seat |
| अरण्य | पुं./न. | forest |

### Changed Meaning by Gender

| Word | Masculine | Feminine/Neuter |
|------|-----------|-----------------|
| नद/नदी | नदः (roarer) | नदी (river) |
| काल/काली | कालः (time) | काली (goddess) |
| कर्म | — | कर्म (n. action) |

## Mnemonic Verses

From Amarakosha:

**पुंल्लिङ्गे**:
> द्यौर्दिग्वह्नेर्दिवसाः...

**स्त्रीलिङ्गे**:
> क्षोणी धात्री...

**नपुंसके**:
> पुष्पं फलं च...

| Abstract (-ता) | Feminine |

| Agent (-तृ) | Masculine |
