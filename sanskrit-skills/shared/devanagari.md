# Devanagari Script Guide

## Vowels (स्वराः)

### Independent Form (used at beginning of words)

| Short | Long | Transliteration |
|-------|------|-----------------|
| अ | आ | a / ā |
| इ | ई | i / ī |
| उ | ऊ | u / ū |
| ऋ | ॠ | ṛ / ṝ |
| ऌ | — | ḷ |
| ए | ऐ | e / ai |
| ओ | औ | o / au |

### Matra Form (attached to consonants)

| Vowel | Matra | Example |
|-------|-------|---------|
| अ | (inherent) | क = ka |
| आ | ा | का = kā |
| इ | ि | कि = ki |
| ई | ी | की = kī |
| उ | ु | कु = ku |
| ऊ | ू | कू = kū |
| ऋ | ृ | कृ = kṛ |
| ॠ | ॄ | कॄ = kṝ |
| ए | े | के = ke |
| ऐ | ै | कै = kai |
| ओ | ो | को = ko |
| औ | ौ | कौ = kau |

## Consonants (व्यञ्जनानि)

### Classified Consonants (वर्गीय व्यञ्जनानि)

Organized by place of articulation (row) and manner (column):

| वर्ग | अघोष अल्पप्राण | अघोष महाप्राण | घोष अल्पप्राण | घोष महाप्राण | अनुनासिक |
|------|----------------|----------------|---------------|---------------|----------|
| कवर्ग (Velar) | क ka | ख kha | ग ga | घ gha | ङ ṅa |
| चवर्ग (Palatal) | च ca | छ cha | ज ja | झ jha | ञ ña |
| टवर्ग (Retroflex) | ट ṭa | ठ ṭha | ड ḍa | ढ ḍha | ण ṇa |
| तवर्ग (Dental) | त ta | थ tha | द da | ध dha | न na |
| पवर्ग (Labial) | प pa | फ pha | ब ba | भ bha | म ma |

### Unclassified Consonants (अवर्गीय व्यञ्जनानि)

| Type | Consonants |
|------|------------|
| **Semivowels** (अन्तःस्थ) | य ya, र ra, ल la, व va |
| **Sibilants** (ऊष्म) | श śa, ष ṣa, स sa |
| **Aspirate** | ह ha |

## Special Signs

### Anusvara and Visarga

| Sign | Name | Sound | Example |
|------|------|-------|---------|
| ं | अनुस्वार | nasal (m̐) | संस्कृतम् = saṃskṛtam |
| ः | विसर्ग | aspiration (ḥ) | रामः = rāmaḥ |

### Chandrabindu

| Sign | Name | Usage |
|------|------|-------|
| ँ | चन्द्रबिन्दु | Nasalized vowel |

### Halanta (Virama)

| Sign | Name | Function |
|------|------|----------|
| ् | हलन्त / विराम | Removes inherent 'a' |

Example: क् = k (no vowel)

## Conjunct Consonants (संयुक्ताक्षराणि)

When consonants combine without intervening vowels:

### Common Patterns

| Conjunct | Components | Example |
|----------|------------|---------|
| क्ष | क् + ष | क्षत्रिय |
| ज्ञ | ज् + ञ | ज्ञान |
| त्र | त् + र | त्रय |
| श्र | श् + र | श्री |
| द्य | द् + य | विद्या |
| द्ध | द् + ध | बुद्ध |
| क्त | क् + त | भक्त |
| ष्ट | ष् + ट | कष्ट |
| ण्ड | ण् + ड | खण्ड |
| न्त | न् + त | अन्त |
| न्द | न् + द | नन्द |
| म्ब | म् + ब | अम्ब |
| ल्ल | ल् + ल | मल्ल |
| त्त | त् + त | सत्त |
| द्द | द् + द | उद्द |

### र-combinations

| Form | Position | Example |
|------|----------|---------|
| र् + consonant | रेफ on top | कर्म, धर्म |
| consonant + र | stroke below | प्र, क्र, त्र |
| consonant + र + vowel | varies | क्रि, प्रा |

## Pronunciation Guide

### Vowel Length
- Short vowels: अ, इ, उ, ऋ (1 मात्रा)
- Long vowels: आ, ई, ऊ, ॠ, ए, ऐ, ओ, औ (2 मात्राः)

### Retroflex vs Dental
- **Dental** (त, थ, द, ध, न): Tongue touches back of teeth
- **Retroflex** (ट, ठ, ड, ढ, ण): Tongue curls back to palate

### Aspirated vs Unaspirated
- **Unaspirated** (क, ग, च, ज, etc.): No breath release
- **Aspirated** (ख, घ, छ, झ, etc.): Strong breath release

### Sibilants
- **श** (śa): Palatal - like "sh" in "ship"
- **ष** (ṣa): Retroflex - tongue curled back
- **स** (sa): Dental - like "s" in "sun"
