# Pratyaya (प्रत्यय) - Suffixes

## Overview

Sanskrit suffixes are broadly categorized as:
- **कृत् प्रत्यय**: Added to धातु (verbal roots)
- **तद्धित प्रत्यय**: Added to प्रातिपदिक (nominal stems)

---

## कृत् प्रत्यय (Primary Derivatives)

### क्त (Past Passive Participle)
**Meaning**: Done, made, having been V-ed

| धातु | + क्त | Meaning |
|------|-------|---------|
| √कृ | कृतम् | done |
| √गम् | गतम् | gone |
| √पठ् | पठितम् | read |
| √दृश् | दृष्टम् | seen |
| √श्रु | श्रुतम् | heard |
| √लिख् | लिखितम् | written |
| √भू | भूतम् | become, been |
| √त्यज् | त्यक्तम् | abandoned |
| √वच् | उक्तम् | spoken |

**Notes**:
- Neuter singular used as abstract noun: गतम् (going, departure)
- Agreement with object in passive: रामेण ग्रन्थः पठितः

### क्तवतु (Past Active Participle)
**Meaning**: One who has done

| धातु | + क्तवतु | Meaning |
|------|----------|---------|
| √गम् | गतवान् | one who went |
| √पठ् | पठितवान् | one who read |
| √खाद् | खादितवान् | one who ate |

**Declines like**: वत्-stem (भगवत्)

### शतृ (Present Active Participle - Parasmaipada)
**Meaning**: V-ing (active)

| धातु | + शतृ | Meaning |
|------|-------|---------|
| √गम् | गच्छत् | going |
| √पठ् | पठत् | reading |
| √हस् | हसत् | laughing |
| √खाद् | खादत् | eating |

**Formation**: Present stem + अत्
**Declines like**: त्-stem (महत्)

### शानच् (Present Active Participle - Atmanepada)
**Meaning**: V-ing (middle)

| धातु | + शानच् | Meaning |
|------|---------|---------|
| √लभ् | लभमानः | obtaining |
| √सेव् | सेवमानः | serving |
| √मन् | मन्यमानः | thinking |

**Formation**: Present stem + मान

### तव्य / अनीय / य (Gerundive/Potential Participle)
**Meaning**: Should be done, to be done, worthy of being done

| Suffix | Example | Meaning |
|--------|---------|---------|
| तव्य | कर्तव्यम् | should be done |
| अनीय | करणीयम् | to be done |
| य | कार्यम् | worthy of being done |
| तव्य | गन्तव्यम् | should be gone |
| अनीय | गमनीयम् | to be gone |
| य | गम्यम् | accessible |

### तुमुन् (Infinitive)
**Meaning**: To V (purpose)

| धातु | + तुमुन् | Meaning |
|------|----------|---------|
| √गम् | गन्तुम् | to go |
| √पठ् | पठितुम् | to read |
| √कृ | कर्तुम् | to do |
| √दा | दातुम् | to give |
| √ज्ञा | ज्ञातुम् | to know |

**Usage**: पठितुं गच्छति (goes to read)

### क्त्वा (Gerund/Absolutive - simple verb)
**Meaning**: Having V-ed (prior action)

| धातु | + क्त्वा | Meaning |
|------|---------|---------|
| √गम् | गत्वा | having gone |
| √कृ | कृत्वा | having done |
| √पठ् | पठित्वा | having read |
| √दृश् | दृष्ट्वा | having seen |
| √खाद् | खादित्वा | having eaten |

### ल्यप् (Gerund - with prefix)
**Meaning**: Having V-ed (prior action, prefixed verbs)

| Verb | + ल्यप् | Meaning |
|------|---------|---------|
| आ+√गम् | आगम्य | having come |
| प्र+√विश् | प्रविश्य | having entered |
| सम्+√भू | सम्भूय | having become |
| नि+√वृत् | निवृत्य | having returned |

**Rule**: Use ल्यप् (→ य) instead of क्त्वा when verb has prefix

### तृच् / तृ (Agent Noun)
**Meaning**: One who does

| धातु | + तृच् | Meaning |
|------|--------|---------|
| √कृ | कर्तृ, कर्ता | doer |
| √दा | दातृ, दाता | giver |
| √गम् | गन्तृ, गन्ता | goer |
| √ज्ञा | ज्ञातृ | knower |
| √श्रु | श्रोतृ | hearer |

### घञ् (Action Noun - masculine)
**Meaning**: Act of V-ing

| धातु | + घञ् | Meaning |
|------|-------|---------|
| √त्यज् | त्यागः | abandonment |
| √भुज् | भोगः | enjoyment |
| √योग् | योगः | union |
| √राज् | राजः | king (one who shines) |

### क्तिन् (Action Noun - feminine)
**Meaning**: Act of V-ing

| धातु | + क्तिन् | Meaning |
|------|---------|---------|
| √कृ | कृतिः | action |
| √गम् | गतिः | motion |
| √मन् | मतिः | thought |
| √भू | भूतिः | prosperity |
| √शक् | शक्तिः | power |
| √सृ | सृतिः | movement |

## तद्धित प्रत्यय (Secondary Derivatives)

### अण् / ण (Patronymic/Belonging to)
**Meaning**: Descendant of, belonging to

| Base | + अण् | Meaning |
|------|-------|---------|
| दशरथ | दाशरथिः | son of Dasharatha |
| वसुदेव | वासुदेवः | son of Vasudeva |
| राम | रामायणम् | relating to Rama |

### मतुप् / वतुप् (Possessive)
**Meaning**: Having, possessing

| Base | + मतुप्/वतुप् | Meaning |
|------|---------------|---------|
| धन | धनवत् | wealthy |
| विद्या | विद्यावत् | learned |
| बुद्धि | बुद्धिमत् | intelligent |
| श्री | श्रीमत् | prosperous |
| भग | भगवत् | blessed, lord |

### इन् (Possessive)
**Meaning**: Having

| Base | + इन् | Meaning |
|------|-------|---------|
| धन | धनिन् | wealthy |
| बल | बलिन् | strong |
| मेधा | मेधाविन् | intelligent |
| तपस् | तपस्विन् | ascetic |
| मनस् | मनस्विन् | wise |

### त्व / ता (Abstract Noun)
**Meaning**: -ness, -ity

| Base | + त्व/ता | Meaning |
|------|----------|---------|
| पुरुष | पुरुषत्वम् | manhood |
| सुन्दर | सुन्दरता | beauty |
| गुरु | गुरुत्वम् | heaviness, guru-hood |
| मधुर | मधुरता | sweetness |
| एक | एकत्वम् | oneness |

### तस् (Ablative Adverb)
**Meaning**: From

| Base | + तस् | Meaning |
|------|-------|---------|
| सर्व | सर्वतः | from all sides |
| अग्र | अग्रतः | from the front |
| पृष्ठ | पृष्ठतः | from behind |

### त्र (Locative Adverb)
**Meaning**: In, at

| Base | + त्र | Meaning |
|------|-------|---------|
| सर्व | सर्वत्र | everywhere |
| अन्य | अन्यत्र | elsewhere |
| एक | एकत्र | in one place |
| तद् | तत्र | there |
| यद् | यत्र | where |

### था (Manner Adverb)
**Meaning**: In such a manner

| Base | + था | Meaning |
|------|------|---------|
| यद् | यथा | as, like |
| तद् | तथा | so, thus |
| अन्य | अन्यथा | otherwise |
| सर्व | सर्वथा | in every way |

### दा (Time Adverb)
**Meaning**: At that time

| Base | + दा | Meaning |
|------|------|---------|
| तद् | तदा | then |
| यद् | यदा | when |
| कद् | कदा | when? |
| सर्व | सर्वदा | always |
| एक | एकदा | once |
