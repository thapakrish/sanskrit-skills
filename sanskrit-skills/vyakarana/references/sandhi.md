# Sandhi (सन्धि) - Euphonic Combination Rules

## Overview

Sandhi = सम् (together) + √धा (to place)

When words or morphemes combine, their junction points undergo phonetic changes for smoother pronunciation.

## Types of Sandhi

1. **स्वरसन्धि** (Vowel Sandhi) - vowel + vowel
2. **व्यञ्जनसन्धि** (Consonant Sandhi) - consonant + consonant/vowel
3. **विसर्गसन्धि** (Visarga Sandhi) - visarga (ः) + following sound

---

## स्वरसन्धि (Vowel Sandhi)

### 1. सवर्णदीर्घ सन्धि (Similar Vowel Lengthening)
**Sutra: अकः सवर्णे दीर्घः (6.1.101)**
Same simple vowels combine into their long form.

| Rule | Example |
|------|---------|
| अ/आ + अ/आ = आ | राम + अवतारः = रामावतारः |
| इ/ई + इ/ई = ई | गिरि + ईशः = गिरीशः |
| उ/ऊ + उ/ऊ = ऊ | गुरु + उपदेशः = गुरूपदेशः |
| ऋ/ॠ + ऋ/ॠ = ॠ | पितृ + ऋणम् = पितॄणम् |

### 2. गुण सन्धि (Guna Sandhi)
**Sutra: आद्गुणः (6.1.87)**
अ/आ + इ/ई/उ/ऊ/ऋ = guna vowel

| Rule | Example |
|------|---------|
| अ/आ + इ/ई = ए | देव + इन्द्रः = देवेन्द्रः |
| अ/आ + उ/ऊ = ओ | सूर्य + उदयः = सूर्योदयः |
| अ/आ + ऋ = अर् | महा + ऋषिः = महर्षिः |

### 3. वृद्धि सन्धि (Vrddhi Sandhi)
**Sutra: वृद्धिरेचि (6.1.88)**
अ/आ + ए/ऐ/ओ/औ = vrddhi vowel

| Rule | Example |
|------|---------|
| अ/आ + ए/ऐ = ऐ | एक + एकम् = एकैकम् |
| अ/आ + ओ/औ = औ | महा + औषधम् = महौषधम् |

### 4. यण् सन्धि (Semivowel Sandhi)
**Sutra: इको यणचि (6.1.77)**
इ/ई/उ/ऊ/ऋ + dissimilar vowel = semivowel + vowel

| Rule | Example |
|------|---------|
| इ/ई + vowel = य् + vowel | इति + आदि = इत्यादि |
| उ/ऊ + vowel = व् + vowel | मधु + अरिः = मध्वरिः |
| ऋ + vowel = र् + vowel | पितृ + आज्ञा = पित्राज्ञा |

### 5. अयादि सन्धि (Diphthong Sandhi)
**Sutra: एचोऽयवायावः (6.1.78)**
ए/ऐ/ओ/औ + vowel = अय्/आय्/अव्/आव् + vowel

| Rule | Example |
|------|---------|
| ए + vowel = अय् + vowel | ने + अनम् = नयनम् |
| ऐ + vowel = आय् + vowel | नै + अकः = नायकः |
| ओ + vowel = अव् + vowel | भो + अनम् = भवनम् |
| औ + vowel = आव् + vowel | नौ + इकः = नाविकः |

### 6. पूर्वरूप सन्धि
**Sutra: एङः पदान्तादति (6.1.109)**
ए/ओ + अ = ए/ओ (अ elided, shown with ऽ)

| Example |
|---------|
| हरे + अव = हरेऽव |
| विष्णो + अव = विष्णोऽव |

### 7. पररूप सन्धि
**Sutra: एङि पररूपम् (6.1.94)**
अ + ए/ओ = ए/ओ (rare, in specific contexts)

## व्यञ्जनसन्धि (Consonant Sandhi)

### 1. श्चुत्व सन्धि
Dental + palatal = both become palatal

| Rule | Example |
|------|---------|
| त् + श् = च्छ् | सत् + शास्त्रम् = सच्छास्त्रम् |
| त् + च् = च्च् | उत् + चारणम् = उच्चारणम् |
| न् + श्/च् = ञ् | तान् + शिष्यान् = ताञ्शिष्यान् |

### 2. ष्टुत्व सन्धि
Dental + retroflex = both become retroflex

| Rule | Example |
|------|---------|
| त् + ट् = ट्ट् | तत् + टीका = तट्टीका |
| त् + ष् = ष् + ट् = ष्ट् | तत् + षष्ठम् = तष्षष्ठम् |
| न् + ट्/ष् = ण् | तान् + षण्ढान् = ताण्षण्ढान् |

### 3. जश्त्व सन्धि
Unvoiced stop + voiced = voiced stop

| Rule | Example |
|------|---------|
| क् + voiced = ग् | वाक् + ईशः = वागीशः |
| त् + voiced = द् | तत् + अन्तः = तदन्तः |
| प् + voiced = ब् | अप् + धिः = अब्धिः |

### 4. चर्त्व सन्धि
Voiced stop + unvoiced = unvoiced stop

| Rule | Example |
|------|---------|
| ग् + unvoiced = क् | वाग् + पतिः = वाक्पतिः |
| द् + unvoiced = त् | तद् + करोति = तत्करोति |

### 5. अनुनासिक सन्धि
Stop + nasal = nasal of same class

| Rule | Example |
|------|---------|
| त् + न् = न्न् | उत् + नतिः = उन्नतिः |
| त् + म् = न्म् | तत् + मात्रम् = तन्मात्रम् |
| क् + न्/म् = ङ् + न्/म् | वाक् + मयम् = वाङ्मयम् |

### 6. ह्-सन्धि
त् + ह् = द्ध्

| Example |
|---------|
| तत् + हि = तद्धि |
| उत् + हारः = उद्धारः |

## विसर्गसन्धि (Visarga Sandhi)

### 1. सत्व (Visarga → स्)
ः + unvoiced dental/sibilant

| Rule | Example |
|------|---------|
| ः + त्/थ् = स्त्/स्थ् | नमः + ते = नमस्ते |
| ः + स् = स्स् | यशः + स्वी = यशस्स्वी |

### 2. रुत्व (Visarga → र्)
अः + voiced (except अ)

| Rule | Example |
|------|---------|
| अः + voiced vowel = र् | शिवः + उवाच = शिवरुवाच |
| अः + voiced consonant = ओ | शिवः + ददाति = शिवो ददाति |

### 3. उत्व (अः → ओ)
अः + soft consonant/अ

| Rule | Example |
|------|---------|
| अः + अ = ओऽ | शिवः + अर्जुनः = शिवोऽर्जुनः |
| अः + voiced = ओ | रामः + गच्छति = रामो गच्छति |

### 4. Visarga retention
ः + क्/ख्/प्/फ् = ः remains

| Example |
|---------|
| रामः + करोति = रामः करोति |
| दुःखम् + पश्यति = दुःखं पश्यति |

### 5. आः Sandhi
आः + voiced = आ (visarga drops)

| Example |
|---------|
| देवाः + गच्छन्ति = देवा गच्छन्ति |

## Internal vs External Sandhi

**Internal (आभ्यन्तर)**: Within a word during derivation
- Compulsory
- Example: गम् + त = गत (not *गम्त)

**External (बाह्य)**: Between words
- Often optional in prose
- Compulsory in verse (for meter)
- Example: राम + अवतारः = रामावतारः or राम अवतारः
